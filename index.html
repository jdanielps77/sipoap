Nuestros Servicios

1Diseño de Sistemas

Estamos en constante desarrollo de sistemas informáticos para optimizar el desempeño de su empresa. Sabemos que cada empresa puede tener requerimientos diferentes, es por esta razón que nuestros sistemas están diseñados de forma que es posible desarrollar cualquier módulo requerido por nuestros clientes, brindando así el sistema mas completo para cada uno de los miembros de nuestra comunidad.

2Asesorías

Brindamos completa asesoría. Pensando siempre en sus necesidades, podemos adaptar y desarrollar sistemas personalizdos según sus requerimientos. Resolvemos sus dudas incluso antes de adquirir nuestros productos. La comunidad Sipoap es lo mas importante para nosotros, es por ello, que estamos presentes en cada uno de sus miembros.

3Soporte Técnico

Una vez esté disfrutando de nuestros productos, podrá contar con servicio técnico personalizado para brindar una mejor experiencia.

4Bases de Datos

Ciertos sistemas requieren alimentación de bases de datos por parte de los usuarios. Podemos facilitarle esa tediosa tarea brindandole una alternativa de alimentación de sus bases de datos para que pueda contar con una información adaptada a sus necesidades.

Nuestros Productos

Análisis de Precios Unitarios V4.0

Nueva versión de este sistema con nuevas herramientas y nueva interfaz gráfica para comodidad del usuario. Adicional a la preparación de un presupuesto completo, este nuevo sistema sistema permite:

- Preparar cómputos métricos por partidas tanto unidades de longitud, área, volumen y peso.

- Creación de cronograma de ejecución (Gantt).

- Preparación de procesos constructivos por cada actividad.

Reportes en formato PDF:

- Presupuestos simples.

- Presupuestos por capítulos.

- Listado de Análisis de Precios Unitarios.

- Mediciones o Memoria de Cálculo.

- Procesos Constructivos de Actividades.

- Resumen General del presupuesto.

- Listado de Materiales.

- Listado de Herramientas y Equipos.

- Listado de Mano de Obra.

Plataforma: Windows. Lenguaje de programación: Python. Bases de Datos: SQlite3

Control de Proyectos y Obras V1.0

Sistema para controlar los avances de las actividades en obras o proyectos. Permite preparar presupuestos por costos directos y a su vez realizar las valuaciones de obra de manera rápida y sencilla.

- Preparar cómputos métricos por partidas tanto unidades de longitud, área, volumen y peso.

Reportes en formato PDF:

- Presupuestos.

- Historial de Avances por cada actividad del proyecto.

- Cuadro de Avances y situación o estatus de actividades en tiempo real.

- Valuaciones de Obra:

Carátula de valuación.

Relación de obra ejecutada.

Cuadro Demostrativo de Obra.

Gráficos.

Plataforma: Windows. Lenguaje de programación: Python. Bases de Datos: SQlite3

Análisis de Precios Unitarios V3.0

Sistema desarrollado totalmente bajo lenguaje de programación Python para Elaborar Presupuestos utilizando la metodología de Análisis de Precios Unitarios, migrado con el objeto de brindar una mejor experiencia de trabajo, optimizando servicios y procesos para obtener mejores resultados, con la sencillez y practicidad de uso que destacan los sistemas SIPOAP. Algunas características son:

- Puedes almacenar diferentes Presupuestos sin limitación de cantidad alguna.

- Almacenamiento de Materiales, Herramientas y Equipos, Mano de Obra, Capítulos y Unidades en una única base de datos independiente, permitiendo así la portabilidad de la misma.

- Creación y Almacenamiento de Análisis de Precios Unitario de manera sencilla y práctica.

- Generación de Reportes en formato PDF.

- Nivel de seguridad PIN por cada proyecto creado.

- Multimoneda.

- Mayor estabilidad que sus predecesoras.

- Servicio técnico y asesoría post venta personalizada.

Plataforma: Windows. Lenguaje de programación: Python. Bases de Datos: SQlite3 y .svc

Análisis de Precios Unitarios V2.0

Segunda entrega de esta herramienta de trabajo, la cual mantiene las mismas características generales de su predecesora pero con funciones adicionales y mejor ambiente de trabajo. Algunas características son:

- Bases de Datos Vacías para ser alimentadas por el usuario.

- Sirve para cualquier tipo de Comercio o Profesional.

- Impresión directa a PDF de Presupuesto, Análisis y Materiales.

- Interfaz de fácil manejo.

- Programado con Macros y Visual Basic Excel 2013.

- Ajustable para cualquier tipo de moneda.

- Almacenamiento ilimitado de Análisis de Precios.

- Clave de Acceso General al Sistema.

Plataforma: Windows. Programación: Visual Basic + Macros Excel

Análisis de Precios Unitarios V0.1.5

Pensado en mejorar y automatizar la Elaboración de Presupuestos, surge este nuevo sistema el cual aplica la Metodología de Análisis de Precios Unitarios para obtener resultados más optimizados y ajustados a la realidad, reduciendo así los margenes de error en precios finales para una oferta de productos o servicios, lo que se traduce en ahorro de dinero para su empresa. Características:

- Sistema compuesto por Plantillas Excel diseñado para la Elaboración y Diseño de Presupuestos de Obras por Análisis de Precios Unitarios.

- Útil tanto para Profesionales como para Empresas.

- Sin requerimiento de Programas adicionales especiales para su manejo, solo abrir el archivo usando Excel 2013 o superior y listo!

- Sirve para cualquier tipo de moneda.

- Base de Datos vacía para ser alimentada por el usuario.

- Ingreso de Materiales, Equipos, Mano de Obra, Capítulos y Unidades en la Base de Datos de Manera Fácil e Intuitiva, según la necesidad del usuario.

- Elaboración de Presupuesto de hasta 50 A.P.U. diferentes.

Plataforma: Windows. Programación: Macros Excel
